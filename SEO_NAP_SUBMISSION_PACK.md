## Workflow для owner-а

1. **Залогинься в pesnopoetsklima@gmail.com** в браузере (одна сессия — все верификации в одном ящике)
2. Открой эту страницу рядом → копируй поля из таблицы NAP
3. Иди по списку Tier 1 → Tier 2 (Tier 3 — потом)
4. Для каждого каталога:
   - Открой "Add business / Добави фирма"
   - Скопируй name, address, phone, email, website **из таблицы NAP** (не пиши руками — опечатки = разбитый NAP)
   - Описание: возьми "Пълно" (BG)
   - Категория: ближайшее к "Климатична техника" / "Климатици"
   - Хорариум: см. таблицу
   - Подтверди email
5. После каждого сабмита — отметь ✅ в списке выше
6. Через 7 дней — проверь индексацию: Google site search `"Песнопоец Клима" -site:pesnopoets-clima.com` (должны появиться backlinks)

## Почему это критично

- Local SEO = NAP + reviews + GBP. Backlinks из BG-каталогов:
  - Дают **citations** (Google сверяет NAP across web — больше совпадений = больше доверия)
  - Дают **referral traffic** (низкий, но реальный)
  - Дают **indexability boost** для GBP в течение 5-дневной верификации
- 8 backlinks из BG-каталогов — это базовый локальный footprint, без него GBP может подвиснуть на верификации дольше
